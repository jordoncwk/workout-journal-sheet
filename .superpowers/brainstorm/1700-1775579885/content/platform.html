Platform Options for Your Mandarin Journal

You need: iPhone support, online + offline access, quick entry

A

Progressive Web App (PWA)

A website that installs on your iPhone home screen like an app. Works fully offline. No App Store needed — just open in Safari and tap "Add to Home Screen". Data syncs to the cloud when online.

Pros

  • Install instantly, no App Store
  • True offline support
  • Free hosting (GitHub Pages etc.)
  • Easy to build and update

Cons

  • Safari has some PWA limitations on iOS
  • Slightly less "native" feel
B

Expo / React Native App

A real native iPhone app built with React Native. Full offline support, native UI components. Distributed via TestFlight (no App Store approval needed for personal use).

Pros

  • Native look and feel
  • Best offline experience
  • Access to device features

Cons

  • More complex setup
  • Requires Xcode + Apple dev account ($99/yr)
  • TestFlight install needed
C

Google Apps Script (like your workout journal)

Same tech as your existing project — runs in Google Sheets, accessible via browser on iPhone. Data lives in Google Sheets automatically.

Pros

  • Familiar — you already have one running
  • Data in Google Sheets (easy to view/export)
  • No hosting setup

Cons

  • No offline support
  • Slow on mobile
  • Not installable as an app

My recommendation: Option A (PWA) — offline + iPhone home screen without any App Store friction. Fastest to build and use.
